# 💰 Wallet Dashboard App

## 📌 Overview

A full-stack wallet dashboard application where users can:

* Add and withdraw money
* View transaction history
* Track balance with charts

---

## 🚀 Tech Stack

* Frontend: React + Tailwind CSS
* Backend: FastAPI
* Database: SQLite (or in-memory)

---

## ⚙️ Features

* Real-time balance updates
* Transaction history (latest first)
* Interactive chart visualization
* Responsive dashboard UI

---

## 🛠️ How to Run

### Backend

cd backend
uvicorn main:app --reload

### Frontend

cd frontend
npm install
npm run dev
